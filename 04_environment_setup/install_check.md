# [교육용 샘플] 개발 및 협업 환경 설치 인증서

*본 문서는 폴리텍대학 AX과정 교육 실습을 위해 개인 노트북 로컬 IT 개발 인프라 환경을 점검하고 기록한 포트폴리오 산출물입니다.*

---

## 1. 로컬 환경 점검 결과 (Codespaces 미사용)
본인의 개인 로컬 컴퓨터 환경에 기 설치 및 신규 구성한 도구 버전 정보는 다음과 같습니다.

* **운영체제(OS)**: Microsoft Windows 11 (NT 10.0.26200.0)
* **Git 설치 버전**: `git version 2.55.0.windows.2`
* **Python 설치 버전**: `Python 3.14.3`
* **Node.js 설치 버전**: `v16.15.0`
* **npm 설치 버전**: `8.5.5`

---

## 2. 설치 장애 및 복구 조치 로그 (선택 사항)
*설치 도중 에러가 나거나 설치에 실패한 학생은 아래 영역에 오류 내용과 캡처 정보를 기록하여 대체 제출하십시오. (오류 해결 노력도 정당한 산출물로 100% 인정됩니다.)*

* **에러 증상**: 터미널에 `python`을 입력하면 버전 대신 마이크로소프트 앱스토어 다운로드 창이 열림.
* **원인 추적**: 윈도우 앱 실행 에일리어스(Alias) 경로 꼬임 현상 발생.
* **디버깅 조치**: 윈도우 설정의 '앱 실행 별칭' 메뉴로 진입하여 python.exe에 할당된 앱스토어 자동 연결 스위치를 해제하여 정상적인 파이썬 버전 출력을 유도함.
* **오류 화면 캡처 백업**: *(해당 없음 - 정상 구성 완료)*
