# [교육용 샘플] DCO 직무 키워드 정의서

*본 문서는 폴리텍대학 AX과정 교육 실습을 위해 가상의 데이터센터 운영 정보를 기초로 작성한 포트폴리오 산출물입니다.*

## 1. DCO 핵심 직무 키워드 5대 요약

1. **하드웨어 트러블슈팅 (Hardware Troubleshooting)**
   - 쉬운 뜻: 컴퓨터 부품인 RAM, SSD, 쿨러 팬이 고장 났을 때 기계를 고치고 부품을 수리하는 행동입니다.
   - 준비 각오: 서버 컴퓨터의 내부 섀시 결합 순서를 꼼꼼히 공부하겠습니다.

2. **광케이블 커넥터 세척 (Fiber Connector Cleaning)**
   - 쉬운 뜻: 데이터 신호가 지나가는 광선 연결 단자의 미세한 먼지나 기름때를 깨끗하게 전용 클리너로 세척하는 절차입니다.
   - 준비 각오: 광 패치 코드 세척을 위한 SOP 규정을 철저히 지키겠습니다.

3. **(여기에 3번째 키워드 입력)**
   - 쉬운 뜻: (내용 입력)
   - 준비 각오: (내용 입력)

4. **(여기에 4번째 키워드 입력)**
   - 쉬운 뜻: (내용 입력)
   - 준비 각오: (내용 입력)

5. **(여기에 5번째 키워드 입력)**
   - 쉬운 뜻: (내용 입력)
   - 준비 각오: (내용 입력)

## 2. 1차시 실습 소감
- 생성형 AI 비서(Gemini)와 직무에 대해 대화해보니, DCO 직무가 어렵고 딱딱한 소프트웨어 코딩에만 국한되지 않고 하드웨어를 꼼꼼히 보살피는 물리적 신뢰성이 중요함을 잘 알게 되었습니다.
